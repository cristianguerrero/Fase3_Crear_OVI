PROGRAMACIÓN WEB 1

En el siguiente OVI de aprendizaje se mostrarán los elementos necesarios, y los pasos a seguir para la vinculación de una base de datos en una página desde PHP con opciones de consulta, edición, modificación y eliminación de datos y/o registros.

Se realizará la instalación del programa XAMPP para la interpretación del lenguaje PHP, y además incluye la herramienta PHPMYADMIN con la intención de manejar la administración de MySQL a través de páginas web.

También se realizará la instalación de un editor de texto para realizar la vinculación desde PHP.

OBJETIVOS DEL OVI

Conocer la instalación del editor de texto para lograr programar por medio de este programar las instrucciones necesarias en PHP para la vinculación.

Explicar la instalación del XAMPP para la interpretación del lenguaje PHP que haya sido programado mediante el editor de texto.

Mostrar la programación y los elementos necesarios para la vinculación de una base de datos en lenguaje MYSQL y una página web programada en PHP para lograr insertar, consultar, modificar y eliminar los datos necesarios.
